
# Haivl-Clone

## Sử dụng:

 - Maven 3.9.6 - Jakarta EE 10 - Servlet
 - Java sdk:22
 - Mysql - com.mysql-connector/j
 - Apache tomcat 10
 - Bootstrap 5.3.3 - jQuery
